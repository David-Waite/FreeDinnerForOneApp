Encryption Pattern

PropertyValue
AlgorithmAES via CryptoJS
Key derivation{userId}-{EXPO_PUBLIC_ENCRYPTION_KEY}
Fallback key"default-secret-key-change-me" (legacy migration)
Payload (sessions)JSON.stringify of the exercises array
Payload (weight)Weight value as a string
Encrypted fielddata (string)
Flag fieldisEncrypted: true
CleanupdeleteField() removes the opposite field on toggle
Encrypted document shape
{
  "id": "1706688000000",
  "name": "Push Day A",
  "date": "2026-01-31T10:30:00",
  "duration": 3240,
  "isEncrypted": true,
  "data": "U2FsdGVkX1+abc123..."  // AES ciphertext
  // exercises field absent
}
Unencrypted document shape
{
  "id": "1706688000000",
  "name": "Push Day A",
  "date": "2026-01-31T10:30:00",
  "duration": 3240,
  "isEncrypted": false,
  "exercises": [ ... ]  // plain array
  // data field absent
}
